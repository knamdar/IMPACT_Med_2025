This repository accompanies the **IMPACT Med 2025** workshop *"Applying MedGemma to Chest X-ray Data"*.  
The session demonstrates how **MedGemma**, a multimodal large language model, can interpret **Chest X-rays** using both image and text reasoning.

Participants learn to generate concise, evidence-based radiology impressions, explore prompt engineering, and discuss the future of **AI-assisted medical imaging**.
